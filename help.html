Generating one-time pad:

  • In the 'seed for pseudo-random one-time pad generator' input box, type in the shared seed you would like to use. This should be agreed upon by both teams in the public message board (in a way that does not let any Saurians know the word).
    • Press 'generate one-time pad'

Enciphering a message:

  • Type in the plaintext message that you would like to send
    • Press 'encipher message'
  • Copy and paste the ciphertext into an email to the public message board board, along with the operation title as the subject.

Deciphering a message:

  • Type in the ciphertext message that you recieved
    • Press 'decipher message'
  • You will need to go through the plaintext message and add spaces and punctuation, because that is all removed in the enciphering process.

REMINDER: Do not re-use one-time pads. Plan ahead to use different words as the seed for each one-time pad.
